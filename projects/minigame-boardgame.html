Boardgame with Minigames


What is this?


This project was an experiment Javascript and Python's Flask Library for a same-screen multiplayer game inspired by Mario Party
It features three minigames.

How does it work?

A standard python Flask app is used to serve web content. The user is given a game number that corresponds to a row in our sqlite3
database. The row contains data about how many coins each player has as well as the state of the board.
