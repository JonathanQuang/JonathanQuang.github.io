Emotion Based Playlist generator


What is this?


This project combines multiple REST APIs and Python's Flask Library to form a lightweight web application that
generates a playlist using a person's lastFM profile as the initial source of songs and picking out songs whose lyrics display a certain emotion

How does it work?

A standard python Flask app is used to serve web content. There is a standard username and password login system that is stored
in an sqlite3 database. The user inputs a lastFM account and selects from a list of emotions to generate a playlist
The process for generating a playlist is detailed below.

  1. We use the lastFM API to fetch a list of songs from the provided link
  2. The titles of the list returned from the lastFM API are used to find the lyrics of the song via the musixMatch API
  3. The first few lyrics of the musixMatch API are passed into Watson's Tone Analyzer to generate a list of emotions per song
  4. The results of this entire process are stored into a DB so that when the user makes another request for a playlist with the same
    last FM account, the whole process described above is not done over again as this will exhaust data limits for the API keys quicker.
    The user does have the option for forcing an update of the stored songs incase new songs are added to the lastFM account
  5. The songs with the proper emotion are displayed on a horizontal slide show using the Youtube API and selecting the first result from it
